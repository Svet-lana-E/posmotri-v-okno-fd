**"Posmotri v okno"** ("look at the window") is a training project for the Yandex Practicum front-end development course.
It's an interactive web page that demonstrates basic HTML, CSS, and JavaScript skills. The project focuses on practicing fundamental techniques: layout, styling, and simple client-side logic.
